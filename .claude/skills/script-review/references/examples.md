# Fact-Checking Examples

## Example 1: GitHub Stars Claim

**Claim in script:** "OpenClaw has 163,000+ GitHub stars"

**Verification process:**
1. Search: "OpenClaw GitHub repository"
2. Check actual star count on GitHub
3. Note the date of verification (star counts change)

**Possible outcomes:**
- ✅ Verified: Link to GitHub repo showing stars
- ⚠️ Outdated: Stars are different, script needs update
- ❌ False: Repository doesn't exist or has far fewer stars

## Example 2: Product Status Claim

**Claim in script:** "OpenClaw has stopped development"

**Verification process:**
1. Check GitHub repository for recent commits
2. Look for official announcements
3. Check if maintainers posted status updates

**Possible outcomes:**
- ✅ Verified: Recent commit says "archived" or "discontinued"
- ⚠️ Nuanced: Development slowed but not stopped
- ❌ False: Active development continues

## Example 3: Feature Capability Claim

**Claim in script:** "Hermes Agent supports 70+ skills"

**Verification process:**
1. Check official documentation
2. Look at the skills/modules directory
3. Count or find official statement about skill count

**Possible outcomes:**
- ✅ Verified: Documentation states "70+ skills"
- ⚠️ Outdated: Actually 85+ now (good problem!)
- ❌ False: Different number or no skills system

## Example 4: Comparison Claim

**Claim in script:** "OpenClaw is #1 AI agent framework"

**Verification process:**
1. What does "#1" mean? Stars? Downloads? Usage?
2. Find the source of this ranking
3. Check if ranking is current

**Possible outcomes:**
- ✅ Verified with caveat: "#1 by stars on GitHub, as of [date]"
- ⚠️ Ambiguous: "#1" is undefined or context-dependent
- ❌ False: No credible source supports this claim

## Red Flag Patterns

### Superlatives Without Qualification
- "The best" → Best by what metric?
- "Most popular" → Most stars? Most downloads? Most mentions?
- "First" → First to do what, specifically?

### Numbers Without Context
- "16万 stars" → As of when? Other projects?
- "70+ skills" → Compared to what's standard?

### Absolute Claims
- "Everyone uses..." → Clearly hyperbolic
- "No other tool can..." → Probably false
- "Never fails..." → Impossible claim

### Competitor Attacks
- "Product X is dead" → Verify, verify, verify
- "Product Y can't do Z" → Check current docs
- "Everyone is switching from..." → Get data
